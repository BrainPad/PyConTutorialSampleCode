# 並列処理と並行処理

## 並列処理とは

処理速度向上を目的として、複数の処理を物理的に同時に実行すること。

## 並行処理とは

リソースの有効活用を目的として、複数の処理を論理的に順不同もしくは同時に実行すること。

# プロセスとスレッド

## プロセス管理とは

プロセスのスケジューリング、割り込み、シグナル制御、優先度の制御、プロセスの切り替え、状態管理、メモリ管理などのOSの重要な役割。

## プロセスとは

プロセスは、CPU上で実行されるものでタスクを完了するためにOSが制御する様々なリソースを使用する。

## スレッドとは

スレッドは、1つのプロセスから生成される実行単位であり同じプロセスから並行でスレッドを起動させる。

# サンプルコード 実行方法

## スレッドベースの並列処理
複数のスレッドを同時に実行

```sh
cd ~/pycon/step1
python multi_threading.py
```

## 並行処理とスレッドプール
複数のスレッドを切り替えながら、擬似的に複数のスレッドを実行

```sh
cd ~/pycon/step1
python thread_pool.py
```
 
# ロギング

##  ロギングとは

プログラムの実行過程を別の記憶装置に記録すること。
Python の標準ライブラリでは、ロギングの用法に便利な関数群を提供している。
詳細は https://docs.python.jp/3/howto/logging.html 参照。
